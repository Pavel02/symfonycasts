Installing Doctrine

Symfony не имеет слоя для работы с БД и делает это через внешнюю библиотеку Доктрина.

Запуск встроенного сервера консольной командой
        php bin/console server:run

Установка пакетов Доктрины
        composer require doctrine

*** Configuring the Database Connection ***
 После установки в файле   .env.dist   добавляется пример для переменной
                    ###> doctrine/doctrine-bundle ###
                    # Format described at http://docs.doctrine-project.org/projects/doctrine-dbal/en/latest/reference/configuration.html#connecting-using-a-url
                    # For an SQLite database, use: "sqlite:///%kernel.project_dir%/var/data.db"
                    # Configure your db driver and server_version in config/packages/doctrine.yaml
                    DATABASE_URL=mysql://db_user:db_password@127.0.0.1:3306/db_name
                    ###< doctrine/doctrine-bundle ###


При установке появился новый конфиг файл            config/packages/doctrine.yaml
 там используется эта переменная.


*** Creating the Database ***
 В файле   .env   указали данные для БД  и теперь БД создать просто одной командой.
                    php bin/console doctrine:database:create
 Создалась пустая БД
            Created database `symfony4_space_bar` for connection named default
