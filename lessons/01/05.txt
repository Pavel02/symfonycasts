Нужно помнить единственное правило контролера: должен возвращаться Symfony Response object.
Абсолютно неважно, как это произойдет: рендеринг шаблона, API ответы, Json ответы.
        Вообще технически контроллер может ничего не возвращать.

Для создания ответа HTML рекомендуется  Twig.
Установка
            composer require twig
 И здесь система рецептов внесет некоторые отличные изменения:
    Изменения в файле   config/bundles.php:
            return [
                Symfony\Bundle\TwigBundle\TwigBundle::class => ['all' => true],
            ];

    Бандлы - это системные плагины для Симфони

 Также рецепт добавил директорию        templates/
 Также добавлен конфиг файл             config/packages/twig.yaml
        Например в этом файле указано, что наши шаблоны по умолчанию в этом каталоге
                twig:
                    paths: ['%kernel.project_dir%/templates']
 Также и др. изменения.
