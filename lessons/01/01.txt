Stellar Development with Symfony4

Уроки по симфони, рекомендуемые офф документацией.
    https://symfonycasts.com/screencast/symfony/setup

Обновляем версию composer
        composer self-update

Команда ниже делает клон проекта symfony/skeleton и запускает
 composer install для скачивания всех зависимостей.
        composer create-project symfony/skeleton the_spacebar

Можем воспользоваться встроенным сервером в PHP.
 Переходим в директорию
        cd the_spacebar
 Запускаем сервер
        php -S 127.0.0.1:8000 -t public


Symfony больше не создает Git репозиторий для Вас, создаем сами.
 Но подготавливает для нас .gitignore , в него нужно добавить файлы phpstorm
        .idea/
 Если требуется, то еще git локально сконфигурировать
        git config --local user.name 'pavellllll'
        git config --local user.email 'test@mail.ru'
 Имея гитигнор, легко создадим свой первый коммит
        git init
        git add .
        git commit -m'add .gitignore'
 Git добавит только около 16 файлов. Стартовое приложение на Symfony - ничего лишнего.
