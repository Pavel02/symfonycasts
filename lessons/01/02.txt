Структура микропроекта Симфони
        /public/index.php   - фронтконтроллер
        /src/           - для кода пользовательского
        /config/        - конфиги

Есть Веб сервер в Симфони. Предназначен для разработки.
        composer require server
Для его запуска
        ./bin/console server:run
 Теперь приложение снова доступно на  http://127.0.0.1:8000


В проекте есть ./bin/console
 Чтобы увидеть ВЕСЬ список команд и возможностей, то вызываем в консоли
        bin/console
